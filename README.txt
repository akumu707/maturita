Syntax:
function:

BLOCK name [*params] {
...
}

!!!Code must begin with a main BLOCK!!!

Commands
WRITE x - print
READ x - input
DO x [*params] - function call
IF bool_exp {
...} - if
WHILE bool_exp {
...} - while
x: exp - assign

!!!Variable names must be lowercase!!!

Supported data types
int
str - quotation marks as an indicator
bool - written with caps - TRUE and FALSE
